# **App Name**: EduStart

## Core Features:

- Interactive Interface: Render an interface component with learning-themed imagery.
- Educational Path Recommendations: AI tool suggests courses and resources for the user based on their profile.
- User Account Creation: Allow users to create personalized accounts.
- Learning Progress Tracker: Visualize the learning progress using dashboards.

## Style Guidelines:

- Primary color: A soft sky blue (#64B5F6) evoking calmness and trust.
- Background color: A very light blue (#E3F2FD), almost white, for a clean feel.
- Accent color: A gentle cyan (#4DD0E1) to add vibrancy without being intrusive.
- Body and headline font: 'PT Sans' (sans-serif) for a balance of modernity and approachability.
- Simple, flat icons to represent different course categories.
- Use asymmetrical layouts with rounded shapes inspired by the image.
- Smooth transitions and loading animations to maintain user engagement.
